# SmartTank
基于STM32的智能坦克
使用STM32标准库，坦克有自动循迹避障模式和遥控模式两种模式。自动循迹避障主要以4路红外循迹和超声波模块配合控制，采用双环PID进行串级闭环调控，内环PID为速度环，用两个PID分别对两个电机的速度进行并行控制，外环PID以循迹信号转换成的状态值作为输入，输出一个速度增量，与基础速度叠加后作为内环PID的输入。在自动循迹的同时通过超声波模块检测前方障碍物距离，距离小于阈值自动刹车，若障碍物靠近则坦克会主动后退避让，障碍物消失后坦克会自动恢复循迹；遥控模式使用HC-04和nRF24L01实现了蓝牙（BLE）和2.4G双模无线通信，BLE通信模式下可用手机等设备作为控制端，2.4G模式下可用下方第二个项目（多功能遥控器）作为控制端。电路使用12V锂电池供给动力电路，并用DC-DC稳压模块（MP1584）稳压后供给控制电路。其余用到的模块：TB6612FNG电机驱动模块等。
